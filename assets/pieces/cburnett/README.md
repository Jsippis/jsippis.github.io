# Cburnett chess pieces

These local SVG assets are Cburnett-style chess pieces generated from the Cburnett piece definitions bundled with TeX Live `mpchess` (`mpchess-cburnett.mp`). They are stored locally so the Smart Chessboard GUI works offline and does not hotlink external image assets.

Attribution: Cburnett / Colin M. L. Burnett.

License note: the bundled TeX Live source file identifies its Cburnett definitions as GPLv2+. The original Cburnett SVG chess pieces are also published on Wikimedia Commons with multiple license options. Keep this attribution/license note with the assets when redistributing the project.
